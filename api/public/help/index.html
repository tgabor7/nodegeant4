Declearing detectors, sources and particle guns

Detectors

The user can describe a detector using the following syntax.

  • name: The name of the detector. Must be enclosed in quotation marks.
  • position: Position of the detector given in x/y/z coordinates. The coordinates should be seperated by commas
  • rotation: Rotation of the detector in euler angles, the angles should be in radians
  • scale: Scale of the detector on the x/y/z axes
  • material: Material of the detector. Must be enclosed in quotation marks.
  • geometry: Shape of the detector. This parameter has to be the name of a volume in the volumes list. Default volume is "cube".
  • Additionally the user can set the unit of measurement for each parameter. Valid units are: mm, cm, dm, m, deg, rad. Default measurement for position and scale are "cm" and "rad" for rotation.

Sources

The user can describe a general particle source using the following syntax.

  • name: The name of the source.
  • position: Position of the source given in x/y/z coordinates. The coordinates should be seperated by commas.
  • material: Material of the source.
  • Additionally the user can set the unit of measurement for the position parameter. Valid units are: mm, cm, dm, m. Default unit of measurement is "cm".

Particle guns

The user can describe a particle gun using the following syntax.

  • name: The name of the particle gun.
  • position: Position of the gun given in x/y/z coordinates. The coordinates should be seperated by commas.
  • direction: The direction in which the gun is aimed at.
  • energy: Energy of the emmited gamma particle.
  • Additionally the user can set the unit of measurement for the position and energy parameters. Valid units are: mm, cm, dm, m for position and eV, keV, MeV, GeV for energy. Default unit of measurement is "cm" for position and "keV" for energy.
